# Inventory Management

This project is a Inventory Management System. It is made with React Native and Node. This project was bootstrapped with **[create-react-app](https://github.com/facebook/create-react-app)**.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
### npm
```
git clone https://github.com/Anubis-spec/inventory-management.git
cd inventory-management
npm install
npm start
``` 
### Yarn
```
git clone https://github.com/Anubis-spec/inventory-management.git
cd inventory-management
yarn install
yarn start
``` 

## Prequisites
1. ```npm``` or ```yarn```
2. ```Node 8.16.0``` or ```Node 10.16.0``` or later version on your local development machine
